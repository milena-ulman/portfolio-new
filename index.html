Workflow

First step

Arranging details with the client, looking for similar websites or applications, drawing inspiration from the projects found.

Second step

Creating a basic representation of a page / application. Choice of colors, fonts, photos. Implementation of the logo and necessary graphics.

Third step

Presenting the first version of the project to the client. Determining design changes.

Fourth step

After obtaining the final client approval, the creation process follows by writing HTML, CSS and JS code.
